Комп’ютерні курси для дітей з пільгових категорій населення #Smile_to_Future

Започатковані у 2017 році. З того часу було безкоштовно надано освітні послуги для понад 300 дітей з комп’ютерної грамотності, основ веб-дизайну, 2-D анімації та різних мов програмування, таких як: "Front-End", "Основ JavaScript, HTLM, CSS, Pyton".

Викладають PRO

На всіх наших курсах викладають тількі класифікованні професіоннали, які мають великий досвіт і мають веику репутацію заздалегіть підготовлені для роботи с дітьми

Безкоштовно для дітей ДЦП

Наші курси безкоштовні для дитини з будь-яким типом інвалідності, всі вчителі заздалегіть підготовлені для роботи с дітьми с інвалідностью та знають як поводи себе в більшойсті випадків

Best Plan

Програма для курсів робилиасья довго та ретельно професіоннали підбирали найоптимальніший варінт щоб було цікаво абсолютно всім дітям без винятка, програм-лист має бути іделаьний
